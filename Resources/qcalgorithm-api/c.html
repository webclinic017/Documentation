Creates a Correlation indicator for the given target symbol in relation with the reference used. The indicator will be automatically updated on the given resolution.

Parameters
Symbol target The target symbol of this indicator.
Symbol reference The reference symbol of this indicator.
Int32 period The period of this indicator.
*CorrelationType correlationType (Optional) Correlation type. Options: ['Pearson', 'Spearman']
*Nullable<Resolution> resolution (Optional) The resolution.
*Func<IBaseData, IBaseDataBar> selector (Optional) Selects a value from the BaseData to send into the indicator, if None defaults to casting the input value to a TradeBar.

Return

Correlation - The Correlation indicator for the given parameters.
