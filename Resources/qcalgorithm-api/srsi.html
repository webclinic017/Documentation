Creates a new Stochastic RSI indicator which will compute the %K and %D.

Parameters
Symbol symbol The symbol whose Stochastic RSI we seek.
Int32 rsiPeriod The period of the relative strength index.
Int32 stochPeriod The period of the stochastic indicator.
Int32 kSmoothingPeriod The smoothing period of K output.
Int32 dSmoothingPeriod The smoothing period of D output.
*MovingAverageType movingAverageType (Optional) The type of moving average to be used.
*Nullable<Resolution> resolution (Optional) The resolution.
*Func<IBaseData, Decimal> selector (Optional) Selects a value from the BaseData to send into the indicator, if None defaults to the Value property of BaseData (x => x.Value).

Return

StochasticRelativeStrengthIndex - A StochasticRelativeStrengthIndex configured with the specified periods and moving average type.
